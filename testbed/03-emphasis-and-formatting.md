# Emphasis and Text Formatting

## Bold Text

This paragraph contains **bold text** using double asterisks.

This paragraph contains __bold text__ using double underscores.

**This entire paragraph is bold.**

**Bold text** at the start, middle, and **end** of a sentence.

Multiple **bold** words in **one** sentence with **bold** formatting.

---

## Italic Text

This paragraph contains *italic text* using single asterisks.

This paragraph contains _italic text_ using single underscores.

*This entire paragraph is italic.*

*Italic text* at the start, middle, and *end* of a sentence.

Multiple *italic* words in *one* sentence with *italic* formatting.

---

## Bold and Italic Combined

This paragraph contains ***bold and italic*** using triple asterisks.

This paragraph contains ___bold and italic___ using triple underscores.

This paragraph contains **_bold and italic_** using mixed markers.

This paragraph contains *__bold and italic__* using mixed markers.

**Bold with *italic inside* bold text.**

*Italic with **bold inside** italic text.*

## Emphasis Edge Cases

**Bold**text (no space)

*Italic*text (no space)

**Bold**text*Italic* (adjacent formatting)

*Italic**Bold**Italic* (nested formatting)

**Bold*Italic**Bold* (complex nesting)

## Emphasis with Numbers

**Bold 123** numbers.

**Bold v1.2.3** version.

**Bold 50%** percentage.

*Italic 123* numbers.

*Italic v1.2.3* version.

*Italic 50%* percentage.

---

## Emphasis with Links

**Bold [link](https://example.com)** text.

*Italic [link](https://example.com)* text.

[**Bold link**](https://example.com) text.

[*Italic link*](https://example.com) text.

---

## Emphasis with Images

**Bold ![image](image.png)** text.

*Italic ![image](image.png)* text.

![**Bold alt**](image.png) text.

![*Italic alt*](image.png) text.

---

## Emphasis with Code

**Bold `code`** text.

*Italic `code`* text.

`**Bold in code**` (should not work).

`*Italic in code*` (should not work).

---

## Emphasis with HTML

**Bold <strong>HTML</strong>** text.

*Italic <em>HTML</em>* text.

<strong>**Bold HTML**</strong> text.

<em>*Italic HTML*</em> text.

## Emphasis Escaping

\*Escaped asterisk\* should not be italic.

\*\*Escaped double asterisk\*\* should not be bold.

\_Escaped underscore\_ should not be italic.

\_\_Escaped double underscore\_\_ should not be bold.
