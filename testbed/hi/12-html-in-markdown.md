# Markdown में HTML

## इनलाइन HTML टैग

इस अनुच्छेद में <strong>HTML मोटा</strong> पाठ है।

इस अनुच्छेद में <em>HTML तिरछा</em> पाठ है।

इस अनुच्छेद में <code>HTML कोड</code> पाठ है।

---

## HTML लिंक

इस अनुच्छेद में <a href="https://example.com">HTML लिंक</a> है।

इस अनुच्छेद में <a href="https://example.com" title="Link Title">शीर्षक के साथ HTML लिंक</a> है।

इस अनुच्छेद में <a href="https://example.com" target="_blank">लक्ष्य के साथ HTML लिंक</a> है।

---

## HTML छवियां

इस अनुच्छेद में <img src="image.png" alt="HTML image"> है।

इस अनुच्छेद में <img src="image.png" alt="HTML image" width="100" height="100"> है।

इस अनुच्छेद में <img src="image.png" alt="HTML image" class="custom-class"> है।

---

## HTML लाइन ब्रेक

इस अनुच्छेद में<br>एक लाइन ब्रेक है।

इस अनुच्छेद में<br/>एक लाइन ब्रेक (स्व-बंद) है।

इस अनुच्छेद में<br />एक लाइन ब्रेक (रिक्त स्थान के साथ) है।

---

## HTML अनुच्छेद

<p>यह एक HTML अनुच्छेद है।</p>

<p>यह एक और HTML अनुच्छेद है।</p>

---

<h1>HTML शीर्षक 1</h1>

<h3>HTML शीर्षक 3</h3>

## Markdown के साथ मिश्रित HTML

इस अनुच्छेद में **Markdown मोटा** और <strong>HTML मोटा</strong> है।

इस अनुच्छेद में *Markdown तिरछा* और <em>HTML तिरछा</em> है।

इस अनुच्छेद में `Markdown कोड` और <code>HTML कोड</code> है।

---

## HTML और Markdown लिंक

इस अनुच्छेद में [Markdown लिंक](https://example.com) और <a href="https://example.com">HTML लिंक</a> है।

---

## HTML और Markdown छवियां

इस अनुच्छेद में ![Markdown छवि](image.png) और <img src="image.png" alt="HTML image"> है।

---

* <code>HTML कोड</code> के साथ आइटम
